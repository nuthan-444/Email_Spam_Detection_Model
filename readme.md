<h1>📧 Spam Email Detection</h1>

<h2>📌 Description</h2>
<p>
This project is a Machine Learning-based Spam Email Detection system that classifies email messages as Spam or Ham (Not Spam). The model uses Natural Language Processing (NLP) techniques to convert text into numerical features using CountVectorizer and trains a Multinomial Naive Bayes classifier to identify patterns commonly found in spam emails. The system can accurately classify new incoming messages and help filter unwanted emails.
</p>

<p><b>Model Accuracy:</b> 98.48%</p>

<h2>🛠️ Tools and Technologies Used</h2>
<ul>
    <li>Python</li>
    <li>Pandas</li>
    <li>NumPy</li>
    <li>Scikit-learn</li>
    <li>CountVectorizer</li>
    <li>Multinomial Naive Bayes</li>
    <li>Joblib</li>
</ul>

<h2>📊 Dataset</h2>
<p>
The dataset contains labeled email messages categorized as Spam and Ham. Text data is transformed into numerical feature vectors using CountVectorizer before being used for model training and prediction.
</p>

<p>
<b>Dataset Link:</b>
<a href="https://www.kaggle.com/datasets/mfaisalqureshi/spam-email">
Kaggle Dataset
</a>
</p>

<h2>📈 Model Performance</h2>
<ul>
    <li>Algorithm: Multinomial Naive Bayes</li>
    <li>Accuracy: 98.48%</li>
    <li>Train-Test Split: 80-20</li>
    <li>Feature Extraction: CountVectorizer</li>
    <li>Classification: Spam / Ham</li>
</ul>
